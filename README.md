# SpaceView Studio

SpaceView Studio is a generative AI web application that creates space-themed artwork from text prompts using diffusion models.

Built as part of a hands-on AI portfolio focused on generative models, backend systems, and deployable ML applications.

---

# Features

* Generate multiple AI images from a single prompt
* Space-themed prompt presets
* Interactive web interface
* FastAPI backend
* Dockerized for reproducibility

---

# Tech Stack

- Python
- FastAPI
- Jinja2
- Hugging Face Diffusers
- PyTorch
- Pillow
- Docker
